Select shipping and billing variants in web shop

The module provides purchase options depending on the method of payment.

If you want to pay in the store when you receive it, then click "Pickup and pay at store". In this case, Proccess Checkout will be shown necessary fields only: name, phone number, email address.

If you want to get the goods in the store, but to pay now, then you can choose "Pickup at store but pay now". Then you will be able to pay online, but it will not show unnecessary fields for shipping information.

If you want to pay now and get a delivery, then you should select "Pay now and get delivery". In this case, the all field will be displayed, so to specify how the online payment, so shipping information.

If you want to pay on delivery, select "Pay on delivery" did not appear to be unnecessary fields of payment and will be shown only those fields that are required to delivery.

The administrator can configure these options enabling or disabling one or some of them are.
